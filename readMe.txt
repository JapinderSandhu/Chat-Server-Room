How to set up/Run:

	Open command prompt and navigate to the directory with the server.js file
	Run the server by typing 'node server.js'
	Open an internet browser and go to 'localhost:2406'
	Open multiple tabs to chat amongst yourself

	To Run:
		type into the chat bar at the bottom to send a message
		double click on a user name to send a private message
		shift+double click on a user name to block them
